## 2. 逐门禁任务

### G3 成本可用化（先做，G2 基线依赖它）

> **实施结果（2026-07-23）**：已完成。`ModelResponse.cost` 现在以 `None` 表示不可用；
> registry 有有效价目且供应商返回 usage 时才估算成本，并记录币种和价目生效日期。同步调用和
> Agent `stream_turn` 都已覆盖，缺价模型不会回填 0。定向模型测试 33 passed，ruff/mypy 通过。

- 在 `config/models.example.yaml` 每个 model 增 `pricing: {input_per_1k, output_per_1k, currency, effective_date}`；
- `registry.py`/`types.py` 解析单价；`gateway.py` 用 usage token × 单价填 `ModelResponse.cost`；
- 无单价的模型仍标 `unavailable`，**不回填 0**；
- 单测：有价模型算出正确成本、无价模型保持 unavailable。
- **产物**：registry 单价字段 + 成本估算单测。**判定**：评测报告成本列不再全 unavailable。

### G1 Planner 可行性评测
